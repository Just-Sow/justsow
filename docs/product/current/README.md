# Current Product Direction

This folder contains the latest agreed product direction for JustSow.

## Current Working Model

JustSow is being built as a product with three main user groups:

- donors who can create accounts and eventually make payments on-site
- Creative Evangelists who submit projects and manage their own content
- employees who manage approvals, publishing, and site oversight

## Internal Workflow Boundary

The website does not need to replace the broader operational workflow that is handled in Notion.

The website should focus on:

- authentication and authorization
- project submission and editing
- approval and revision decisions
- publishing state
- donor/payment flows
- auditability of key actions
