# Roadmap

## Phase 1: Repo Foundation

Completed as part of the monorepo and Turbo workflow rollout.

- root monorepo structure
- standardized workspace tooling
- root developer workflow
- initial shared package boundary

## Phase 2: Identity And Access

- define multi-role user model
- implement account creation and sign-in
- implement email verification and password reset
- design account claiming for manually created sower records
- implement two-factor authentication

## Phase 3: Project Workflow

- implement project creation and draft flow
- implement gatekeeper review workflow
- implement project revision and resubmission flow
- implement moderated post-approval project edits
- implement project publishing and case study flow

## Phase 4: Sowers And Seeds

- implement sower accounts and history views
- implement manual sower creation by staff
- implement one-seed-per-project workflow
- implement seed allocation and audit records

## Phase 5: Production Readiness

- finalize deployment workflow
- establish monitoring and error tracking
- finalize infrastructure for early production
- complete security and operational runbooks

## Phase 6: Post-MVP Expansion

- implement on-site payment processing
- expand donor account capabilities
- revisit infrastructure for broader geographic distribution
- evaluate deeper reporting and operational integrations
